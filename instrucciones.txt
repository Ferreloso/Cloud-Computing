Solo para los devs

❗ Para la primera vez que haces clone al repositorio:
    1. ejecuta ▶️npm install◀️ para instalar las dependencias que usa el proyecto

🟠 Para correr el servidor:
    1. primero usa el comando ▶️npm run build◀️ para generar tu carpeta con los archivos compilados
    2. una vez hecha la build, corre tu servidor de desarrollo con ▶️npm run start◀️
